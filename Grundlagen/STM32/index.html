# STM32

STM32 ist eine Mikrocontroller-Familie von ST mit einer 32-Bit ARM Cortex-M0/M3/M4 CPU.

Der STM32 besitzt kein EEPROM für die persistente Speicherung von Daten weshalb auf einen externes IC zurückgegriffen werden muss.

# Getestete Boards
